# Networking Project README

## About the Project

This project was developed using Cisco Packet Tracer for Haile Hotels and Resorts located in Debre Birhan. The purpose of the project is to design and simulate a network that can support the hotel's daily operations, including communication between departments, internet access, and secure network management. The network is designed for more than 500 users and covers multiple floors and departments within the hotel.

## Project Title

Fundamentals of Networking

## Course Information

**Course Title:** Fundamentals of Networking

## Project Objectives

The main objectives of this project are:

* To design a complete network for a hotel environment.
* To divide departments using VLANs.
* To enable communication between VLANs through inter-VLAN routing.
* To provide automatic IP address assignment using DHCP.
* To implement OSPF dynamic routing.
* To configure NAT for internet connectivity.
* To secure remote access using SSH.
* To provide wireless connectivity for the restaurant section.

## Features Implemented

The network includes the following features:

* Hierarchical network design (Core, Distribution, and Access layers)
* VLAN configuration for different departments
* Inter-VLAN routing using Layer 3 switches
* DHCP server for automatic IP address allocation
* OSPF routing protocol
* NAT/PAT configuration for internet access
* SSH for secure device management
* Access Control Lists (ACLs) for additional security
* Wireless network support for restaurant users

## Tools Used

* Cisco Packet Tracer 8.2.2
* Networking configuration and simulation tools

## Project Files

* **project.pkt** – Cisco Packet Tracer simulation file
* **Networking document.docx** – Project report and documentation

## Network Devices Used

The network topology contains:

* 1 ISP Router
* 1 Border Router
* 2 Core Layer 3 Switches
* 3 Distribution Switches
* 7 Access Switches
* Servers
* PCs
* Wireless Access Points

## Security Measures

To improve security, the following configurations were applied:

* VLAN separation between departments
* SSH for secure remote administration
* ACLs to limit unauthorized access
* DHCP for centralized IP management
* Static IP addresses for important network devices

## How to Run the Project

1. Open the **project.pkt** file using Cisco Packet Tracer.
2. Read the **Networking document.docx** file for detailed explanations and configurations.
3. Test the VLANs, routing protocols, DHCP, NAT, SSH, and other network services implemented in the project.

## Conclusion

This project demonstrates the practical application of networking concepts learned in the Fundamentals of Networking course. It shows how a secure, scalable, and well-organized network can be designed and implemented for a real-world hotel environment using Cisco Packet Tracer.
